## 项目说明

next-intl-starter 是一个基于 Next.js 的启动模板，内置国际化、暗黑模式等功能。它为构建现代 Web 应用提供了坚实的基础，并内置了多语言支持。

## 功能特点

- 🌍 **国际化**: 使用 next-intl 实现多语言支持
- ⚡️ **现代技术栈**: 基于 Next.js 15 App Router，提供最佳性能
- 📦 **包管理**: 使用 pnpm 实现更快更高效的包管理
- 🎨 **样式方案**: 集成 Tailwind CSS，快速开发 UI
- 🌓 **暗黑模式**: 内置暗色/亮色主题切换
- 📱 **响应式**: 采用移动优先的设计方案

## 快速开始

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev
```

## 项目结构

```text
├── src/
│   ├── app/               # Next.js 应用目录
│   ├── components/        # 可复用组件
│   └── i18n/              # 国际化配置
├── messages/              # 翻译文件
├── public/                # 静态资源
```
